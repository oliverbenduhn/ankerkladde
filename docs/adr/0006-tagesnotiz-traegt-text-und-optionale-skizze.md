# Tagesnotiz trägt Text und optionale Skizze

**Status**: accepted

Text und Tages-Skizze gehören zum selben datumsgebundenen Item der Journal-Kategorie; das Item entsteht beim ersten Speichern eines der beiden Inhalte. Ein eigener Tagesblatt-Typ oder ein separates Skizzen-Item würde zusätzliche Zuordnung und Lebenszykluslogik schaffen, ohne einen eigenständigen fachlichen Nutzen zu liefern.
